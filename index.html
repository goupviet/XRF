Cutting Room Floor XRF

Linear, retraux, frame-by-frame video filtering and editing.

Learn More

Retraux (adj.) - a portmanteau of retro and faux

Frame by frame editing

Cutting Room Floor allows you to edit videos frame by frame, applying filters and various other graphic manipulation techniques to each each frame in sequence. Edited frames can be extracted in full resolution, or committed back to the movie file. Audio streams will not be touched.

Custom Filters

Don't like the included filters? No offence will be taken *, just re-write them. XRF includes a built-in filter editor.

* Well maybe a little bit..

Use the F(Fmpeg)orce, Luke...

XRF is powered by FFmpeg, and as such supports most if not all common video formats. Additionally, the movie file can be converted to any format or resolution that FFmpeg supports. After all, XRF is just a big collection of FFmpeg commands (don't tell anyone, shh).

It will Blend

Made in C# .NET, using Windows Presentation Foundation, XRF's UI can be changed and re-written using Microsoft® Blend, or even just writing pure XAML code.

But what's it good for?

It sounds pretty niche doesn't it? Editing a video frame by frame. But there's a surprising number of uses:

  • Minor video touch up - You don't need a ~£1000 editing package just to clean up a bit of footage, but Movie Maker might not be the best choice either. XRF could probably do the job nicely. And with a shorter encoding time.
  • Censoring - Let's leave that there.
  • Instagram-style filters - Instagram's great, but maybe you'd like to apply a similar filter to a video longer than 15 seconds. And not have to make it square. We got you covered.
  • GIFs - Tumblr will love this one. Extract frames from a video, save it as a GIF! We have that! XRF does the thing!
  • Hacking the mainframe - This is a more developy one, but you can use XRF's source code freely in your own projects. After all, we're licensed under GNU GPL v3.0, it'd be rude not to. This particular bullet point leads us nicely on to...

DEVELOPERS, DEVELOPERS, DEVELOPERS

XRF is fully open-source, with extensive documentation including VS2013 Solution files, unit tests, file format specifications, and XML comment-generated API documentation. All source code files (including XAML files) are fully documented, so hack away.
